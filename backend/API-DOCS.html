🛍️ Shopster API Documentation

🔐 JWT Токени

Де брати токен:
  • POST /api/auth/login або POST /api/auth/register
  • Отримаєте accessToken (7 днів) та refreshToken (30 днів)
Заголовок Значення
Authorization Bearer YOUR_ACCESS_TOKEN

📦 Orders - ВАЖЛИВО!

Щоб створити замовлення:
  1. Спочатку додайте товари в кошик (POST /cart)
  2. Потім створіть замовлення (POST /orders)
Автоматично береться: user з токену, товари з кошика, ціна
МетодURLАвторизаціяОпис
GET/orders/myBearerМої замовлення
POST/ordersBearerСтворити (тіло тільки shippingAddress!)

🔄 Workflow (приклад)

// 1. Логін
const { accessToken } = await fetch('/api/auth/login', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ email, password })
}).then(r => r.json());

// 2. Додати в кошик
await fetch('/api/cart', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': 'Bearer ' + accessToken
  },
  body: JSON.stringify({ productId: '...', quantity: 1 })
});

// 3. Створити замовлення (кошик очиститься автоматично!)
await fetch('/api/orders', {
  method: 'POST',
  headers: { Authorization: 'Bearer ' + accessToken },
  body: JSON.stringify({ shippingAddress: { ... } })
});
